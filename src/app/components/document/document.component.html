Installation

Prerequisites

This section will give you some information about what tools you will need. Bellow part will describe all the prerequisites and how to install the app.

To install and use Angulary you will need to install Angular CLI, NodeJs, npm package manager and git globaly in your computer.

Download and install NodeJs form the website.

Download and install the latest Git from its website.

Open (Terminal, Command Prompt etc.) and run:

                    
npm install -g @angular/cli
